# 製品目的

## 製品名

親子で育てるいきもの研究所

## 対象利用者

- 小学生の子ども
- 保護者
- 親子で一緒に使う家庭

## 製品目的

- 生き物への興味を引き出す。
- 抽選・発見・収集を入口にする。
- 興味を持った生き物を研究情報へつなげる。
- 親子の会話を促す。

## 優先する体験

引く → 新しい生き物を見る → 名前、画像、レア度を認識する → もう一度引く → 気になったときに研究情報を開く。

## 製品として行わないこと

- 読むことを強制する。
- 研究情報を抽選の障害にする。
- 子ども向けを理由に内容を空疎にする。
- GUI上の都合だけでゲーム原則を変える。
- 将来構想を現在実装済みのように扱う。

個別のレイアウトやCSS仕様は[GUI共通原則](GUI_PRINCIPLES.md)の対象とする。
