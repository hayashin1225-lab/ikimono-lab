# ゲーム原則

## 中心ループ

引く → 新しいカードが出る → 絵、名前、レア度を見る → すぐ次を引く。

## 基本原則

- 生き物の発見を主要な報酬とし、研究は中心ループを妨げない。
- 読むことを強制しない。
- 未取得が残っている間は未取得を優先する。ただし永久的な重複禁止ではなく、全取得後は既存候補へフォールバックする。
- 最少 `pages` 優先と `lastId` 回避を維持する。
- 特別研究は通常抽選と混同しない。
- 将来構想と現在仕様を分ける。

## 現在の将来構想

以下は確定実装ではない。通常カードの上限、ジャンル別パック、生き物とジャンルの組合せ、特別研究チケット、特別研究専用抽選、個別カード画像、学年別文章を検討対象とする。

製品目的は[PRODUCT.md](PRODUCT.md)、GUIの共通原則は[GUI_PRINCIPLES.md](GUI_PRINCIPLES.md)を正本とする。
